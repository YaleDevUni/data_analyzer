Fetch Weather Data

Fetching Weather Data: Implemented a Django view function fetch_weather_data that retrieves weather data from the weather API based on a country. The data is fetched using the requests library, parsed, and saved to the database using the Weather model.

Displaying Weather Data: Created the weather.html template that renders weather information such as location, temperature, wind, humidity, etc. The fetched weather data is passed to the template as weather_data using the render function in the fetch_weather_data view.

Data Visualization

Data Visualization:Have created a Django view function visualize_data that fetches weather data from the database using the Weather model. The data is then converted into a pandas DataFrame for easy manipulation and visualization.

Plotting: Using Matplotlib, Have plotted the relationship between temperature and UV index. The temperature values are represented on the x-axis, and the UV index values are represented on the y-axis. Each data point is visualized as a scatter plot, with the color set to 'skyblue'. Additionally, I have added a regression line to show the overall trend in the data.

Real-time Data Analyze and Documentation

Correlation Analysis: Perform a correlation analysis between the temperature (in Celsius) and the UV index variables. The correlation coefficient is calculated using pandas, indicating the strength and direction of the linear relationship between the variables. The correlation result is presented in a table format.

Hypothesis Testing: Set up the hypothesis for testing the correlation between temperature and UV index. The null hypothesis assumes no correlation, while the alternative hypothesis suggests a correlation. The hypothesis test aims to provide evidence to either reject or fail to reject the null hypothesis based on the calculated p-value.
